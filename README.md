# Моделирование системы искусственной вентиляции лёгких

## Что исследовали?

Моделирование работы системы искусственной вентиляции лёгких (ИВЛ) при различных состояниях дыхательной системы.

Построение и сравнение трёх режимов вентиляции для прогнозирования параметров дыхательного цикла.

**Тип исследования:** вычислительный эксперимент  
**Размер:** 3 режима, 3 параметра оценки

## Как исследовали?

Построили математическую модель дыхательной системы на основе уравнения движения:

$$
P = P_{\text{peep}} + \frac{V}{C} + R \cdot Q
$$

**Where:**
- $P$ = Total pressure
- $P_{\text{peep}}$ = Positive end-expiratory pressure
- $V$ = Volume
- $C$ = Compliance
- $R$ = Resistance
- $Q$ = Flow (often denoted as $\dot{V}$)

Вход системы: Q
Выходное значение: P 

Провели численное моделирование для каждого режима. Оценили параметры дыхательного цикла: объём, давление, расход.

**Параметры модели:**

| Параметр | Значение | Описание |
|----------|----------|----------|
| C | 0.05 | Эластичность лёгких (л/см H₂O) |
| R | 2.0 | Сопротивление (см H₂O·с/л) |
| Vt | 0.5 | Дыхательный объём (л) |
| PEEP | 5 | Давление в конце выдоха (см H₂O) |

## Результат исследования:

Проведено 3 эксперимента с различными параметрами системы.

| Режим | Объём (л) | Давление (см H₂O) | Расход (л/мин) |
|-------|-----------|-------------------|----------------|
| Норма | 0.525 | 12.5 | 30.0 |
| Повышенное R | 0.489 | 15.2 | 25.1 |
| Сниженная C | 0.396 | 18.7 | 34.8 |

**Ключевые выводы:**

- **Нормальный режим:** объём 0-0.525 л, давление 5-12.5 см H₂O — физиологическая норма
- **Повышенное сопротивление:** давление увеличивается на 22%, объём снижается на 7%  
- **Сниженная эластичность:** давление растёт на 50%, объём падает на 25%
- **Главный фактор:** изменение эластичности и сопротивления значительно влияет на работу дыхательной системы
- **Взаимодействие факторов:** при сочетании сниженной эластичности и повышенного сопротивления давление может превысить 20 см H₂O
